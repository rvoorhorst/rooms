# Rooms

A fully featured Discord-like chatroom web app. Main features:

- Chatrooms
- Topics
- Search
- Registration and login
- User profiles
- Direct messages


## Requirements

* Python v3.9.x
* Django==3.2.13


## Install

```
pyenv virtualenv 3.9.10 rooms
pyenv activate rooms
make install
```

## Run

```
python manage.py runserver
```
